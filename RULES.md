# User Rules (Priority)

이 파일에 작성한 규칙을 최우선으로 반영해 작업합니다.

## How to use
1. 아래 `## Rules` 구역에 규칙을 추가/수정하세요.
2. 규칙을 우선 반영해야 하는 경우, 이 파일만 업데이트하면 됩니다.

## Rules
- [ ] 규칙 1: (예) 앞으로 수정/생성 파일에는 항상 짧은 한국어 주석을 1~2줄 넣기
- [ ] 규칙 2: (예) 테스트/검증 절차가 있으면 마지막에 체크리스트로 포함하기
- [ ] 규칙 3: 앞으로 코드를 바로 적용하지 말고 검증할 수 있도록 처리하기
- [ ] 규칙 4: SOLID 원칙을 항상 준수하며 코드를 작성하기
- [ ] 규칙 5: 디자인패턴/리팩터링으로 더 나은 구조가 가능하면 적극적으로 제안하기

## Notes
- 규칙이 충돌하면, 더 구체적인 규칙이 우선됩니다.
- 기간/범위(예: “이 대화 동안”, “전체 프로젝트”)가 필요하면 규칙에 명시해 주세요.
